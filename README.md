# 🤖 AI Resume Analyzer


## About Project

AI Resume Analyzer is a Flask-based web application that analyzes resumes using NLP techniques.

The application extracts resume content, identifies skills, calculates a resume score, and provides improvement suggestions.


## Features

- Upload Resume PDF
- Extract Resume Text
- AI Skill Analysis
- Resume Score Calculation
- Missing Skill Detection
- Learning Recommendations


## Technologies Used

- Python
- Flask
- PyPDF2
- Scikit-learn
- HTML
- CSS


## How To Run


Install requirements:

pip install -r requirements.txt


Run application:

python app.py


Open:

http://127.0.0.1:5000
